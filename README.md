# Trello Slack Overdue

Send all your Trello overdue tasks to a Slack channel

## How to install
1. `git clone https://github.com/CosmicResearch/trello-slack-overdue.git`
2. Move slack-example.json to slack.json and trello-example.json to trello.json
3. Put your Slack app's hook and the destination channel in slack.json
4. Put your key, token and org name in trello.json
5. Done

You can run it periodically with a cron job.
